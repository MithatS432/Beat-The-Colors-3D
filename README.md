# Beat-The-Colors-3D
Our goal is to manage the colors and complete the puzzles.

# Game Explanation
It is a game where we collect colored balls that come to us in an infinite loop. After the 3rd wave, it starts to get confusing, we gain a feature in every 5 waves, we can get the ball we want for 5 seconds, and after the 7th wave, the color texts change every 5 seconds, making the game more challenging.

# Controls
A or D to movement and Space to jump
